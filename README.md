**Swarm Robot BT** is a ROS 2 and MuJoCo-based framework for LLM-assisted Behavior Tree planning and decentralized multi-robot coordination.

The current system is developed and evaluated in a MuJoCo multi-robot simulation environment using 9 ROBOTIS OpenMANIPULATOR-X style robots. In the next stage, the framework will be validated on physical ROBOTIS OMX robots to test whether the same BT-based coordination structure can be transferred from simulation to real hardware.
